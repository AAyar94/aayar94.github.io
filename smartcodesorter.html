About this plugin

SmartCodeSorter

SmartCodeSorter is a Kotlin code-organizing plugin for IntelliJ IDEA and Android Studio. It automatically sorts class members, constructor parameters, and more — keeping your Kotlin code clean, consistent, and easy to read without any manual effort.


Shortcut: Option + Command + S (macOS)
Or navigate to Code → Smart Code Sorter → Sort Kotlin Code



Features


Sort Class Members Automatically sorts fields, methods, and nested classes within Kotlin classes following a consistent, readable ordering.

Constructor Parameter Sorting Organizes constructor parameters in a logical order, making your class definitions easier to understand at a glance.

Sort on Save Enable automatic sorting every time you save a file via Code → Smart Code Sorter → Sort on Save. Never think about member ordering again.

IntelliJ IDEA & Android Studio Compatible with IntelliJ IDEA and Android Studio. Integrates seamlessly into your existing workflow with no extra setup required.

Keyboard Shortcut Trigger sorting instantly with Option + Command + S on macOS, or via the Code menu on any platform.

Zero Configuration Install and go. SmartCodeSorter works out of the box with sensible defaults that follow Kotlin coding conventions.

How to Use

Manual Sort
  1. Open any Kotlin file in IntelliJ IDEA or Android Studio.
  2. Press Option + Command + S (macOS).
  3. Or go to Code → Smart Code Sorter → Sort Kotlin Code.

Sort on Save
  1. Go to Code → Smart Code Sorter → Sort on Save.
  2. Toggle the option on — your code will be sorted automatically on every save.
