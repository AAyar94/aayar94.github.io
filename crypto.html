Crypto Tracker

Live cryptocurrency tracker with coin details, price charts, and an adaptive multi-pane layout for phones and tablets.

Kotlin Compose MVI Retrofit
All Projects

Overview

Modern Android
best practices

Built as a reference project to explore Android's latest best practices. Crypto Tracker pulls live coin data and displays it with a clean list-detail UI. On tablets, it uses Adaptive Navigation for a proper multi-pane experience.

  • Live cryptocurrency list with price & change data
  • Detail view with price chart per coin
  • Adaptive Navigation — single/dual pane based on screen size
  • MVI architecture for predictable state management
  • Koin for lightweight dependency injection
